## About

This project is part of a tutorial series from Net Ninja, where I'm learning to build a React Native application using Expo and Expo Router.

## Project Details

- **Framework**: React Native with Expo
- **Router**: Expo Router
- **Language**: TypeScript

## Getting Started

To run this project:

```bash
# Install dependencies
pnpm install

# Start the development server
pnpm start

# Run on specific platform
pnpm ios      # iOS simulator
pnpm android  # Android emulator
pnpm web      # Web browser
```
